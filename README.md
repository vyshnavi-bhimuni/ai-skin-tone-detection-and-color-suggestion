# AI-Based Skin Tone Detection and Color Suggestion

## Overview

This project is an AI-based application developed using Python. It detects a user's skin tone and recommends suitable clothing colors based on predefined skin tone and color datasets. The application uses color mapping techniques to provide personalized color suggestions.

---

## Features

- Detects skin tone
- Recommends suitable clothing colors
- Uses RGB color mapping
- Dataset-based color recommendation
- Simple and user-friendly application

---

## Technologies Used

- Python
- Pandas
- NumPy
- CSV Datasets

---

## Dataset Files

- **color_rgb_map.csv** – RGB color mapping data
- **skin_tone_dataset.csv** – Skin tone dataset
- **suitable_colors.csv** – Clothing color recommendations

## How to Run

1. Install the required libraries.

```bash
pip install pandas numpy
```

2. Run the application.

```bash
python color.py
```

3. The application detects the skin tone and displays suitable clothing color recommendations.
